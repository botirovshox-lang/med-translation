# 🔘 Кнопки Segment Editor — Руководство

## Обзор кнопок в таблице

Каждая строка сегмента содержит 14 колонок с кнопками действий:

| № | Колонка | Описание | Действие |
|----|---------|---------|----------|
| 0 | ✓ | Select segment | Выбрать сегмент для просмотра в правой панели |
| 1 | ID | Номер сегмента | (только чтение) |
| 2 | Source | Исходный текст | (только чтение) |
| 3 | Target | Целевой перевод | Редактируемое поле (auto-save) |
| 4 | TM% | Процент совпадения TM | (только чтение) |
| 5 | 🔍 TM | Find TM | Поиск в Translation Memory |
| 6 | ▶️ GPT | Translate (OpenAI) | Перевод через OpenAI с глоссарием |
| 7 | 🌐 Google | Translate (Google) | Перевод через Google Cloud API |
| 8 | ✓ QA | QA Check | Проверка качества перевода |
| 9 | ⤴️ Back | Back-check | Обратный перевод для проверки |
| 10 | Status | Статус сегмента | Текущее состояние (new, translated, qa_done, needs_review, confirmed, etc.) |
| 11 | QA | QA Score | Результат QA (0-100) |
| 12 | ✅ | Confirm & Save | **Подтвердить перевод и добавить в TM** |
| 13 | ❌ | Clear Translation | **Очистить перевод и сбросить статус** |

---

## ✅ Кнопка "Confirm & Save"

### Когда использовать
- После завершения перевода сегмента
- Когда QA проверка пройдена успешно (статус `qa_done`)
- Когда все изменения внесены в Target поле

### Что происходит при клике
1. ✅ Проверяется, что в Target поле есть текст
2. 📝 Статус сегмента меняется на `confirmed`
3. 💾 Перевод **автоматически добавляется в Translation Memory (TM)**
4. 🔄 Страница обновляется, сегмент выбирается в правой панели
5. 📢 Выводится сообщение: "✅ Saved & Added to TM!"

### Результат в БД
- `segments.status` = `'confirmed'`
- Запись добавляется в таблицу `translation_memory`:
  - `source_hash`: SHA256 хеш исходного текста
  - `source_text`: Исходный текст
  - `target_text`: Переведённый текст
  - `domain`: `'medical'`

## ❌ Кнопка "Clear Translation"

### Когда использовать
- Если перевод неправильный и нужно начать заново
- Если сегмент нужно переделать
- Для отката ошибочного перевода

### Что происходит при клике
1. 🗑️ Target поле **полностью очищается** (становится пустым)
2. 🔄 Статус сегмента сбрасывается на `new`
3. 💾 Кеш сессии очищается для данного сегмента
4. 📢 Выводится сообщение: "🗑️ Cleared"
5. 🔄 Страница обновляется

### Результат в БД
- `segments.target_text` = `''` (пусто)
- `segments.status` = `'new'`
- Сегмент готов для пересчёта

### ⚠️ Важно
- **Операция необратима** — очищенный перевод нельзя восстановить
- Если перевод был в TM, он остаётся в TM
- При очистке сегмента с `confirmed` статусом, новый перевод не будет в TM до повторного подтверждения

## 🔄 Полный цикл работы сегмента

```
1. new (исходное состояние)
   ↓
2. Нажимаем одну из кнопок перевода (▶️ GPT или 🌐 Google)
   ↓
3. translated (перевод получен)
   ↓
4. Опционально: нажимаем 🔍 TM или ▶️ / 🌐 ещё раз для улучшения
   ↓
5. Нажимаем ✓ QA для проверки качества
   ↓
6. qa_done (если QA прошла) или needs_review (если найдены проблемы)
   ↓
7. Если needs_review → исправляем перевод, нажимаем ✓ QA заново
   ↓
8. Нажимаем ✅ Confirm & Save
   ↓
9. confirmed (финальное состояние + добавлено в TM)
```

## Session State (кеширование)

Приложение использует Streamlit session state для оптимизации:

- `st.session_state.target_cache[seg_id]`: Кеш значений Target для быстрого обновления UI
- `st.session_state.selected_segment_id`: ID выбранного сегмента (для правой панели)
- `st.session_state.current_page`: Текущая страница в пагинации

### Автоматическая синхронизация
- Когда вы редактируете Target → автоматически сохраняется в БД
- Когда нажимаете кнопку действия → кеш обновляется перед rerun()

## 💡 Советы

1. **Для 100% совпадения**: Нажимайте 🔍 TM перед перевод**ом — если совпадение 100%, оно вставится автоматически
2. **Для качественного перевода**: Используйте оба переводчика (GPT и Google) и выбирайте лучший
3. **Перед подтверждением**: Всегда запускайте ✓ QA чтобы избежать ошибок
4. **Back-check**: Используйте ⤴️ для проверки семантического значения на сложных сегментах
5. **Изменение статуса**: При любом клике на кнопку действия статус меняется автоматически
