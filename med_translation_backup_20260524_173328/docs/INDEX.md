## 🔧 Модули приложения

| Модуль | Файл | Роль |
|--------|------|------|
| **UI** | `app_v55.py` | Streamlit интерфейс (основное приложение) |

| **DOCX** | `docx_cat.py` | Импорт/экспорт DOCX файлов |

| **OpenAI** | `openai_client.py` | Обёртка для OpenAI API |
| **Модели** | `schemas.py` | Pydantic модели для QA отчётов |
| **Prompts** | `prompts.py` | LLM prompts для перевода и QA |
| **Очистка** | `cleanup_db.py` | Скрипт очистки HTML-сущностей |

## ❓ Часто задаваемые вопросы

**Q: С чего начать?**  
A: Прочитайте STATUS_REPORT.md и CONTEXT.md

**Q: Как запустить приложение?**  
A: `streamlit run app_v55.py` → http://localhost:8501

**Q: Где моя документация по старой версии?**  
A: В папке `docs/` есть все файлы (включая устарелые для справки)

**Q: Где сохраняются переводы?**  
A: В SQLite БД в папке `data/` (segments table)

**Q: Как добавить в Translation Memory?**  
A: Нажмите кнопку ✓ (toggle button) для confirm

**Q: Почему Google Translate не работает?**  
A: Нужно активировать биллинг в Google Cloud Console
