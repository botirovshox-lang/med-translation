# 📊 Excel-Like Table with ag-Grid

**Дата**: 2026-05-24  
**Статус**: ✅ РЕАЛИЗОВАНО

## 🎯 Что это?

**ag-Grid** - профессиональная Excel-like таблица с полным функционалом:
- ✅ Resize колонок (drag & drop)
- ✅ Сортировка (click на заголовок)
- ✅ Фильтрация (панель сбоку)
- ✅ Inline редактирование (Target колонка)
- ✅ Регулируемая высота
- ✅ Горизонтальный scroll
- ✅ Темная тема

---

## 🚀 Как установить

```bash
pip install streamlit-aggrid
```

Или обнови requirements.txt и выполни:
```bash
pip install -r requirements.txt
```

## 🎮 Как использовать

### **1. Resize колонок**

```
Наведи мышь на границу между заголовками:
┌──── Source ────┌──── Target ────┐
                 ↑ Cursor меняется на ↔️
                 
Drag влево/вправо → колонка меняет ширину
```

✅ **Работает как в Excel!**
- Drag & drop границы между колонками
- Минимальная ширина: 50px
- Максимальная ширина: не ограничена

---

### **2. Сортировка**

```
Click на заголовок колонки:
Source ← Click
  ↓
Source ↑ (ascending) ← Click again
  ↓
Source ↓ (descending)
```

✅ **Сортирует данные в реальном времени**

---

### **3. Фильтрация**

```
Слева есть панель фильтров:
☰ Status ▼  (выбери: new, translated, confirmed, etc.)
☰ QA ▼      (выбери: пусто, 50-75, 75-100)
```

✅ **Фильтрует таблицу на лету**

---

### **4. Редактирование Target**

```
Click на ячейку в колонке Target:
[editable] ← Click
  ↓
[cursor появляется]
  ↓
Типируй текст → Автоматически сохраняется в БД
```

✅ **Inline edit - как в Google Sheets**

---

### **5. Регулируемая высота**

```
Сверху есть слайдер:
Table height (px) 🎚️ 200 ←→ 800

Потяни слайдер → таблица становится больше/меньше
```

✅ **Всё в одной страничке - не нужно скроллить вниз**

## 💾 Как сохраняется данные

### **Target редактирование**

```
Шаг 1: Кликнешь на Target ячейку
Шаг 2: Типируешь текст
Шаг 3: Нажимаешь Enter или уходишь из ячейки
  ↓
Данные АВТОМАТИЧЕСКИ сохраняются в SQLite БД
  ↓
DB обновляется без перезагрузки страницы
```

✅ **Сохранение работает в фоне**

## ⚙️ Настройки ag-Grid

В коде можно менять:

```python
# Минимальная ширина колонок
min_column_width=50

# Высота таблицы
height=table_height  # 200-800px слайдер

# Количество строк
grid_data = []
for seg in segs[:100]  # Максимум 100 сегментов

# Можно добавить:
# - Pagination (постраничная выдача)
# - Search (поиск)
# - Export CSV (экспорт)
# - Custom styling (кастомные цвета)
```
