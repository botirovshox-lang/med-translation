## Изменения в config/переменные

**Переменные окружения:**
```powershell
$env:ANTHROPIC_API_KEY = 'sk-ant-...'  # для build_glossary.py
$env:PYTHONUTF8 = '1'                   # всегда UTF-8 на Windows
```

**Конфиги в коде (легко менять):**
```python
# download_pdfs.py
BASE_URL = "https://medlineplus.gov/languages/russian.html"
DELAY = 1.2  # сек между скачиванием PDF

# extract_tm.py
BATCH_SIZE_WORDS = 8  # слова за раз при группировании строк
NOISE_RE = re.compile(...)  # фильтрация шума

# build_glossary.py
MODEL = "claude-haiku-4-5-20251001"
BATCH_SIZE = 8  # пар за API-запрос
DELAY = 0.5  # сек между запросами
```

**Внешние сервисы:**
- MedlinePlus: https://medlineplus.gov/languages/russian.html
- Google Cloud Storage: healthinfotranslations-pdfdocs (автоматически через ссылки)
- Anthropic API: https://console.anthropic.com/ (требует API key)
