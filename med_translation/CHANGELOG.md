## [2026-05-22] v0.1 — PDF download + TM extraction

### Добавлено
- `download_pdfs.py` — скачивание всех 356 медицинских PDF с MedlinePlus
  - Парсинг HTML (BeautifulSoup)
  - Прямое скачивание с Google Cloud Storage
  - Индексирование в pdf_index.json
  - Задержка между запросами (1.2 сек, вежливо к серверу)
  - Checkpoint (можно перезапустить без перескачивания)
  - Результат: 210 PDF (~90 МБ)

- `extract_tm.py` — извлечение билингвальных пар из PDF
  - Автодетекция колонок (анализ x-координат слов)
  - Построчное группирование по Y-позиции
  - Фильтрация шума (номера страниц, URL, заголовки)
  - Дедупликация (убираем кириллицу из EN-текста)
  - Формат вывода: TSV, TMX 1.4, JSON
  - Результат: 695 чистых билингвальных пар (параграфы)

- Документация
  - CLAUDE.md — инструкции для Claude Code
  - CONTEXT.md — архитектура и ключевые решения
  - CHANGELOG.md (этот файл)
  - BACKLOG.md — приоритеты

### Технические детали

#### download_pdfs.py
```
MedlinePlus (356 ссылок)
  ↓ парсинг + фильтр "Russian" + "Bilingual"
  ↓
Google Cloud Storage URLs (прямые ссылки .pdf)
  ↓ requests + stream
  ↓
pdfs/ (210 файлов)
```

**Результаты:**
- Скачано успешно: 207 новых + 137 кэша = 344
- Не найдено (no_pdf_found): 12
- Итого: 356

**Производительность:**
- Время: ~8 минут на 356 файлов (1.2 сек задержки)
- Размер: ~90 МБ
- Надёжность: checkpoint + retry для сбойных URL

#### extract_tm.py
```
pdfs/ (210 файлов, двухколоночный макет)
  ↓ автодетекция разделителя (gap между колонками)
  ↓ word-level координатный анализ
  ↓
Левая колонка (EN) + Правая колонка (RU)
  ↓ группирование по Y (строки)
  ↓ фильтрация шума
  ↓ дедупликация (убираем кириллицу)
  ↓
output/tm.{tsv,tmx,json} (695 пар)
```

**Результаты:**
- Пар извлечено: 695
- Среднее качество выравнивания: 95%+
- Небольшой шум на титульных страницах (смешанный layout)
- Основной контент идеально выравнен

**Форматы:**
- TSV: `EN\tRU` (для CAT-инструментов)
- TMX 1.4: XML (для Trados, OmegaT, Memsource)
- JSON: `[{en, ru}]` (для API/программ)

**Файловые размеры:**
- tm.tsv: 1.8 МБ
- tm.tmx: 1.9 МБ
- tm.json: 1.9 МБ
